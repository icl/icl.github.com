About

Summary

Hypertext Markup Language (HTML) and Cascading Style Sheet (CSS) are two simple yet powerful languages for building web sites. They make possible an implementation of the concepts and theories of design discussed in this class; solid understanding of their syntax and roles is a requisite of fully understanding and working with these concepts. If you have no experience with these languages, or need more of a review than provided here, please visit the Cognitive Science 25 modules: XHTML lecture set, CSS lecture set.

In this lecture we will review the material at high speed.

Goals

  • to review principle concepts in XHTML and CSS
  • to introduce these concepts to beginners

You Will Learn

  1. how to build XHTML forms and form elements
  2. how to build XHTML tables, and nested tables
  3. implementing CSS styles and classes
